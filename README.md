# Twitter-like API (Golang)

This project is a backend API that simulates basic features of a social media platform like Twitter.

## Features
- User creation
- Post tweets
- Fetch tweets
- Basic API structure

## Tech Stack
- Golang
- REST APIs
- JSON

## API Endpoints
- POST /users
- POST /tweets
- GET /tweets

## Purpose
This project was built during my internship to understand scalable API design and backend structuring.
